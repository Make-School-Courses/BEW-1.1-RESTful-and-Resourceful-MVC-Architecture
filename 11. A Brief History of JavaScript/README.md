## What is JavaScript?

### Multi-Paradigm

JavaScript is **Multi-Paradigm** meaning it is a flexible language that can be used with various programming paradigms, namely:

* Procedural/Imperative
* Object Oriented/Classical
* Prototypical
* Functional
